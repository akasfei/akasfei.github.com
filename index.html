    • INTRODUCTORY SEQUENCE

      What is SFEI Systems ISN?

      I-S-N stands for Imaging Sequential Network. It is associated with SFEI Systems, which is named after its author, SFEI. It is a website which stores SFEI's hightlighted projects as well as a brief introduction of himself.

      ISN features a sequantial layout, including sequence groups, sequences, and sequence nodes. Sequences with different catagories are arranged in different sequence groups and each sequence represents a specific topic. A sequence consists of several nodes, which represents articles explaining the corresponding topic.

      You can use arrow keys, enter and backspace to navigate through the nodes and sequences.

    • OPERATING REFERENCE

      KEYBOARD SHORTCUTS

      You can use arrow keys, enter and backspace to navigate through the nodes and sequences. Specific key navigations are:

      When a sequence node is active, use and keys to navigate through other nodes. Use and keys to move to the next or previous sequence. Use ENTER to active the default button (the blue one), and BACKSPACE to move up to the parent sequence.

      When a sequence is active, use and keys to navigate through other sequences. Use and keys to activate the last or first node in this sequence. Use ENTER to active the first node, and BACKSPACE to move up to the parent sequence group.

      When a sequence group is active, use and keys to navigate through other groups. Use ENTER to active the first sequence in this group.

    • SFEI SYSTEMS v3

      SUMMARY

      SFEI Systems v3, codenamed North Star, is an online blog website built with Node.js and Mongodb. It features a twitter-like follow/follower relationship, and SFEI's design concept of 'Imaging Database Network'. It is designed to be the host for SFEI's blogs, thoughts, references, etc.

      NOTE to enter the HUST JingJing website design contest, SFEI applied some minor modifications to this website, including rebranding the website to 'North Star' (the codename) and adding many archive entries associated with the contest. In version 3.3.2, the website will be rebranded to SFEI Systems.

      Currently SFEI Systems v3 is hosted on Cloudfoundry. The source code is available on GitHub.

    • SFEI SYSTEMS v3

      PURPOSE AND INTENTIONS

      Inpsired by the intro movies of Call of Duty Modern Warfare series, an idea of building a private blog website with the name of 'Imaging Database Network' came to mind. I decided to call it 'SFEI Systems', which is also the name of a website I've created before. The project 'SFEI Systems Mk III Imaging Dataabase Network' later became the prototype of the award-winning North Star.

      NOTEThe codename of SFEI Systems is 'North Star', but North Star is also the name of a branch of this website. North Star is codenamed 'Deep Earth' and won first place of the HUST JingJing website design contest that took place in Oct 2012 to Jan 2013.

      SFEI Systems is my first website available to general audience of the Internet. This website will mainly consist of my blogs, thoughts, references, etc. Also recently I've been thinking to convert it into an API server, thus I can access the website's data from multiple applications and platforms.

    • SFEI SYSTEMS v3

      FUTURE ASPECTS

      SFEI will keep maintaining this website for a considerable amount of time.

      In version 3.3.2:

      • the webiste will be rebranded to its orginal form, SFEI Systems;
      • some small bugs will be fixed;
      • admins will be able to remove unused tags.

      In future versions, 3.4.x or 3.5.x, the website will include an API, which will enable viewing, editing, commenting articles using AJAX. For example, ISN will be able to use a js plugin to fetch new projects from SFEI Systems.
